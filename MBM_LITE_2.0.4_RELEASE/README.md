\# MBM-LITE



Lightweight incremental backup manager for Windows.



\## Features



\- Incremental file backup

\- Direct folder synchronization

\- Pause / Resume jobs

\- Error handling

\- Lightweight GUI

\- No restore points

\- No disk images

\- No proprietary archive containers




\## Installation



Download the latest installer from Releases.



\## Requirements



Windows 10/11
